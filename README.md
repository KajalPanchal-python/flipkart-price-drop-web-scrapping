# 🛒 Flipkart Price Drop Web Scraping Project

## 📌 Project Overview
The Flipkart Price Drop Web Scraping Project is a Python-based application designed to monitor product prices on Flipkart. The project collects product information, tracks price changes, and helps users identify price drops efficiently.

---

## 🚀 Features
- Extract product details from Flipkart
- Track product prices
- Store product information in a database
- Interactive web interface
- Display product details and price updates

---

## 🛠️ Technologies Used
- Python
- HTML
- CSS
- JavaScript
- SQLite Database
- Web Scraping

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/KajalPanchal-python/flipkart-price-drop-web-scrapping.git
```

2. Open the project folder:

```bash
cd flipkart-price-drop-web-scrapping
```

3. Install required libraries:

```bash
pip install beautifulsoup4 requests
```

4. Run the project:

```bash
python main.py
```
